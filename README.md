Запустити цей проект з об'єднаним бекендом (Django) та фронтендом (React + Vite) на Windows можна за кілька кроків. Ось покрокова інструкція для локального запуску:

1. Підготовка та скачування проекту

1.1. Встановіть Python, Node.js та Git, якщо вони ще не встановлені.

2.1. Відкрийте командний рядок (Terminal, CMD або PowerShell) та клонуйте репозиторій:

bash

git clone <посилання_на_ваш_github_репозиторій>

cd <назва папки проекту>

2. Налаштування та запуск Backend (Django)

2.1. Створіть та активуйте віртуальне оточення:

bash

python -m venv venv

venv\Scripts\activate


2.2. Встановіть залежності та застосуйте міграції:

bash

pip install -r requirements.txt

python manage.py migrate

2.3. Запустіть сервер Django:

bash

python manage.py runserver

3. Налаштування та запуск Frontend (React + Vite)

3.1. Відкрийте нове вікно терміналу, перейдіть до папки з фронтендом (зазвичай вона називається frontend або client) та встановіть пакети:

bash

cd <папка з фронтендом>

npm install

3.2. Запустіть сервер розробки Vite:

bash

npm run dev

Сервер буде доступний за адресою:

http://localhost:5173/ (вкажіть порт із терміналу).
